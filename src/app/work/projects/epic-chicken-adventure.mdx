## Overview

A browser-based arcade game built with Next.js 14 and React.

## Features

- Character movement and collision detection
- Progressive difficulty system
- Score tracking with high scores
- Responsive controls for mobile and desktop

## Tech Stack

- **Framework**: Next.js 14, React 19
- **Styling**: Tailwind CSS
- **Game Logic**: Canvas rendering and collision detection

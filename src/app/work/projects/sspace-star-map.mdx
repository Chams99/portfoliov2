## Overview

An interactive star map application integrating Stellarium Web Engine with NASA/JPL astronomical data.

## Features

- Interactive star map with Stellarium Web Engine
- Star search by registration codes
- Personal star naming and collection system
- LocalStorage-based data persistence
- Smooth zoom and navigation controls

## Tech Stack

- **Engine**: Stellarium Web Engine
- **Data**: NASA/JPL astronomical datasets
- **Storage**: LocalStorage for personal collections
- **Frontend**: JavaScript for API integration

This project demonstrates the power of modern web technologies in making astronomy accessible and interactive. Users can explore the cosmos from their browser, learn about different celestial objects, and create their own meaningful connections with the stars.

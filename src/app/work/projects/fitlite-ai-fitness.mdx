## Overview

A fitness app landing page with dynamic theme switching and glassmorphism design. Built with vanilla HTML/CSS/JavaScript, featuring smooth animations and mobile-first responsive design.

## Key Features

- Dynamic dark/light theme switching
- Real-time image updates with theme changes
- Glassmorphism design elements
- Smooth animations and transitions
- Mobile-first responsive design
- LocalStorage theme persistence

## Technical Implementation

### Frontend Architecture
- **HTML5**: Semantic markup and accessibility
- **CSS3**: Custom properties, flexbox, and grid
- **JavaScript**: ES6+ features and DOM manipulation
- **Tailwind CSS**: Utility-first styling framework

### Theme System
- CSS custom properties for dynamic theming
- LocalStorage for preference persistence
- Real-time image switching logic
- Smooth transition animations

### Performance Optimization
- Optimized images and assets
- Efficient CSS and JavaScript
- Lazy loading implementation
- Mobile performance optimization

## Development Challenges

- **Theme Switching**: Implementing smooth transitions between light/dark modes
- **Image Management**: Dynamic image switching based on theme
- **Performance**: Maintaining smooth animations on mobile devices
- **Responsive Design**: Ensuring glassmorphism effects work across all screen sizes

## Overview

Dent is a professional dental clinic website that showcases premium dental care services with a modern, user-friendly interface. Built with Next.js 14 and TypeScript, this project demonstrates advanced web development skills with a focus on performance, accessibility, and user experience.

## Key Features

- **Complete Dental Services**: Comprehensive dental care including teeth whitening, Invisalign, cosmetic dentistry, and dental implants
- **Fully Responsive Design**: Optimized for all devices from mobile to desktop
- **Next.js Performance**: Leveraging Next.js 14 for optimal loading speeds and SEO
- **Modern Professional UI**: Clean, elegant design that builds trust with patients
- **Appointment Booking System**: Integrated scheduling functionality
- **Patient Testimonials**: Social proof with 5-star ratings
- **Contact Integration**: Multiple ways for patients to get in touch

## Technical Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Cloud Platform
- **Performance**: 95+ Lighthouse score

## Challenges & Solutions

### Challenge 1: Performance Optimization
**Solution**: Implemented Next.js Image optimization, code splitting, and lazy loading to achieve excellent Core Web Vitals scores.

### Challenge 2: Professional Design
**Solution**: Created a trust-building design with clean layouts, professional imagery, and intuitive navigation that resonates with healthcare patients.

### Challenge 3: Accessibility
**Solution**: Ensured WCAG 2.1 compliance with proper ARIA labels, keyboard navigation, and screen reader support.

## Results

- ✅ Professional healthcare website that builds patient trust
- ✅ Excellent performance metrics (95+ Lighthouse score)
- ✅ Fully responsive across all devices
- ✅ Modern, maintainable codebase with TypeScript

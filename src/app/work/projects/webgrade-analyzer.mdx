## Overview

A website performance analyzer built with Google Lighthouse API integration.

## Features

- Google Lighthouse API integration
- Mobile and desktop performance analysis
- Core Web Vitals monitoring
- Accessibility and SEO audits
- Security vulnerability detection
- LocalStorage scan history

## Tech Stack

- **API**: Google Lighthouse API
- **Frontend**: JavaScript, LocalStorage
- **Metrics**: Performance, Accessibility, SEO, Security
